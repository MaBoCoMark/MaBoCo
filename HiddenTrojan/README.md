# A method of Nginx + Apache Server.
> May 16 2022, Added PHP WebProxy

## How to find config path
```
Prefer to use Find Fuction by:
find / -name httpd.conf
```
### Nginx config file path
```
/etc/nginx/
```
### Apache config file path
```
/etc/httpd/conf/
```
### Htdocs path
```
/var/www/html
```
